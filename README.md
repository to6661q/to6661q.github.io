# to6661q Portfolio

A dynamic, high-performance personal portfolio engineered by an RPA Developer. Features seamless integration with Google Sheets (acting as a headless CMS via Apps Script) and the GitHub REST API for real-time project showcasing.

## 🛠 Tech Stack
- **Core:** React 18, Vite, JavaScript
- **Styling:** Tailwind CSS, Glassmorphism UI
- **Animation:** Framer Motion
- **RPA Data Pipeline:** Google Apps Script (REST API endpoint)
- **External API:** GitHub REST API (Repository metrics)
- **Deployment:** GitHub Pages

## ⚡ Key Features
- **RPA-Driven CMS:** Career, license, and contact data dynamically fetched and parsed from Google Sheets via secure REST endpoints.
- **Live GitHub Integration:** Auto-fetches pinned repositories, programming languages, and real-time metrics (stars/forks).
- **Performance Optimized:** Responsive architecture, smooth scroll animations, and optimized asset loading.
